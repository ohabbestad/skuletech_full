📖 Kva er boolean?

Ein boolean er ein verdi som berre kan vere eitt av to ting: True (sant) eller False (usant). Slike verdiar kjem ut av samanlikningar:

print(10 > 5)   # True
print(10 < 5)   # False
print(10 == 10)  # True
💡 = vs ==: Eitt likskapsteikn (=) tildeler ein verdi til ein variabel. To likskapsteikn (==) samanliknar to verdiar og gjev True eller False.

📋 Samanlikningsoperatorar

🔀 if / else / elif

Med if kan Python ta avgjerder og gjere ulike ting avhengig av kva som er sant:

tal = int(input("Skriv eit tal: "))
if tal > 0:
    print("Positivt")
elif tal < 0:
    print("Negativt")
else:
    print("Null")
if
Sjekkar det første vilkåret
elif
Sjekkar neste vilkår viss det første var usant
else
Køyrer viss ingen av dei andre var sanne

🎯 Oppgåver

  1. 1 Lag eit program som spør etter alderen til brukaren og skriv ut om han er «barn» (under 13), «ungdomsskulelev» (13-15) eller «vidaregåandeelev» (16 og eldre).
  2. 2 Bruk input() til å lage eit enkelt «stemme eller ikkje»-program: viss alderen er 18 eller meir, skriv «Du kan stemme!», elles skriv ut kor mange år som manglar.
  3. 3 Kombiner if med ei for-løkke: gå gjennom tala 1-20 og skriv «Partal» eller «Oddeleg» for kvart tal. (Tips: bruk modulo-operatoren %, t.d. tal % 2 == 0 er sant for partal.)
  4. 4 Kombiner while med if: lag eit gjetteprogram der brukaren prøver å gjette eit tal mellom 1 og 10. Viss svaret er for høgt, skriv «For høgt!», viss for lågt, skriv «For lågt!», viss rett, avslutt løkka.
  5. ★ Ekstra: Lag ein enkel kalkulator som spør etter to tal og ein rekneoperasjon (+, -, *, /) og skriv ut svaret. Bruk if/elif for å skilje mellom operasjonane.
