For-løkker

for i in range(n)

📺 Sjå videoen

📖 Kva er ei for-løkke?

Ei løkke er ein måte å seie «gjer dette fleire gonger» utan å skrive den same linja om og om att. For-løkka bruker nøkkelordet for saman med range():

for i in range(5):
    print("Hei!")

Viktig: innrykk!

Koden som skal gjentakast, må ha 4 mellomrom (eller ein tabulator) som innrykk. Python bruker innrykket til å forstå kva som er inni løkka og kva som er utanfor.

📖 Tre måtar å bruke range()

range(n), frå 0 til n-1

Tel frå 0 og opp til (men ikkje med) n:

for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

range(x, y), vel start og slutt

Start på x, tell opp til (men ikkje med) y:

for i in range(1, 6):
    print(i)  # 1, 2, 3, 4, 5

range(x, y, d), vel steglengda

Hopp d av gongen. Kan óg gå baklengs med negativt tal:

for i in range(0, 11, 2):
    print(i)  # 0, 2, 4, 6, 8, 10

for i in range(10, 0, -1):
    print(i)  # 10, 9, 8, ... 1

🔢 Rekne med løkker

For-løkker er svært nyttige i matematikk. Her summerer vi alle tal frå 1 til 10:

sum = 0
for i in range(1, 11):
    sum = sum + i
print("Summen er:", sum)  # → 55

For kvar runde legg vi til den gjeldande verdien av i på summen. Eit klassisk mønster!
