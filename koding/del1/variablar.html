Variablar

Del 1, Kva er ein variabel?

Kva er ein variabel?

Ein variabel er som ein minneboks i datamaskina. Du gjev boksen eit namn, og legg ein verdi inni. Seinare kan du hente ut verdien att berre ved å bruke namnet.

namn = "Ola"
alder = 13

print(namn)    # → Ola
print(alder)   # → 13

Legg merke til at namn har hermeteikn fordi det er tekst. alder derimot har ikkje hermeteikn fordi det er eit tal. Python behandlar desse ulikt!

Del 2, Byggje setningar med variablar

Setningar med variablar

Du kan kombinere tekst og variablar inne i print() ved å bruke komma eller f-streng. Den enklaste måten er komma, Python set då inn eit mellomrom automatisk.

namn = "Ola"
alder = 13

print("Hei,", namn)    # → Hei, Ola
print(namn, "er", alder, "år gamal")    # → Ola er 13 år gamal
