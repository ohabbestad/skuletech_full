Kva er datatypar?

Python skil mellom ulike typar verdiar. Eit tal er ikkje det same som tekst, sjølv om dei ser like ut. Dei tre viktigaste datatypane er:

int
Heiltal
10, 34, -5
float
Desimaltal
3.14, 9.81, 1.5
str
Tekst (string)
"Ole", "Hei!"

Tekst (str) har alltid hermeteikn rundt seg. Tal (int og float) har ikkje det.

alder = 34      # int
karakter = 4.5   # float
namn = "Ole"   # str

type(), sjå kva type noko har

Du kan bruke den innebygde kommandoen type() til å sjekke kva datatype ein variabel har:

print(type(34))       # <class 'int'>
print(type(4.5))     # <class 'float'>
print(type("Ole"))   # <class 'str'>

Kvifor dette er viktig

Hermeteikn endrar korleis Python behandlar verdien. Sjå her:

print(10 + 5)     # → 15 (reknar)
print("10" + "5")  # → 105 (set saman tekst!)
💡 Når Python ser hermeteikn, trur han at det er tekst, og set då bokstavar etter kvarandre i staden for å rekne. Tal utan hermeteikn, tekst med hermeteikn.
