Kva er input()?

Til no har programma dine alltid gjort det same. Kvar gong du køyrer koden, kjem same svaret ut. input() let deg lage program som "snakkar med brukaren".

namn = input("Kva heiter du? ")
print("Hei,", namn)

Teksten inni input() er meldinga brukaren ser på skjermen. Det brukaren skriv, kjem inn i variabelen namn.

input() gjev alltid tekst

Sjølv om brukaren skriv inn eit tal, vil input() alltid returnere ein str (tekst):

tal = input("Skriv eit tal: ")
print(type(tal))  # <class 'str'>

Vil du bruke svaret i eit reknestykke, må du gjere det om med int() (for heiltal) eller float() (for desimaltal):

alder = int(input("Kor gamal er du? "))
neste_ar = alder + 1
print("Neste år er du", neste_ar, "år gamal.")

Oppgåver

  1. 1 Køyr koden og skriv inn namnet ditt. Kjem det korrekt «Hei, [namn]» tilbake?
  2. 2 Legg til ein ny input() som spør etter favorittidretten. Bruk svaret i ei setning.
  3. 3 Lag eit program som ber om to tal frå brukaren og skriv ut summen. Hugs å bruke int()!
  4. 4 Lag eit program som ber om eit tal og reknar ut det dobbelte. Skriv ut: «Det dobbelte av X er Y.»
  5. ★ Ekstra: Lag eit lite spørjeskjema med minst tre spørsmål (namn, alder, favorittfag). Skriv ut ei oppsummering til slutt: «[namn] er [alder] år og likar [fag].»
